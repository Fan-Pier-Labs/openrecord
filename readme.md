# MyChart Connector

A MyChart OpenClaw plugin and MyChart MCP server.

- **OpenClaw Plugin**: Self-contained plugin that bundles all MyChart scraper code locally. Registers 35+ tools for accessing medical records, medications, lab results, messaging, and more — no server needed.
- **MCP Server**: Per-user MCP server for Claude AI integration. Connect your MyChart accounts and access your health data through Claude.
